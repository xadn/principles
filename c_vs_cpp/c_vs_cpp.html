5 Differences between C and C++


You don't have to declare functions before using them

int sum(int a, int b); // You don't need this in C int main() { int c = sum(1, 2); return 0; } int sum(int a, int b) { return a+b; }
http://www.cprogramming.com/tutorial/c-vs-c++.html

There is no boolean type

The following code will cause an error in C:
bool visited = false; // this doesn't work in C

Use an enum to workaround this:
typedef enum {true, false} bool;
http://www.cprogramming.com/tutorial/c-vs-c++.html

Deallocating memory is simpler

C++ code:
int *x = new int; // allocate memory int *x = new int[10]; delete x; // free memory delete[] x;

C code:
int *x = malloc( sizeof(int) ); // allocate memory int *x_array = malloc( sizeof(int) * 10 ); free( x ); / free memory free( x_array );
http://www.cprogramming.com/tutorial/c-vs-c++.html

There are no classes

Unfortunately, there is no way to encapsulate methods like you can in a class. At least you can bundle data variables together in a struct.
struct kitten { char* name; int lovability; } sam, max;

Add a typedef so you can declare new kittens wherever you want!
typedef struct { char* name; int lovability; } kitten; kitten ralph; ralph.lovability = 20;
http://cprogramminglanguage.net/c-structure.aspx

C doesn't have cout or stream operators

Instead of:
cout << "I love C++";

Do this:
printf("but, C is waaaaaay better");
